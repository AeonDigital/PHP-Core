Instalação
----------

Instale em seu projeto usando o composer:


**Via terminal**

.. code-block:: console

    composer require aeondigital/phpcore


**Via composer.json**

.. code-block:: json

    "require": {
        "aeondigital/phpcore": "dev-master"
    }
