==================
mb_str_starts_with
==================


.. php:function:: mb_str_starts_with( $haystack, $needle)
	
		.. rst-class:: phpdoc-description
		
			| Identifica se a ``string`` passada inicia com a sentença indicada.
			
		
		
		:Parameters:
			- ‹ string › **$haystack** |br|
			  ``String`` original.
			- ‹ string › **$needle** |br|
			  Caracteres iniciais.

		
		:Returns: ‹ bool ›|br|
			  Retornará ``true`` se ``$haystack`` é uma ``string`` que inicia com ``$needle``.
